Chat Bubble Overlay

Tampilan chat Twitch kustom berbentuk gelembung. Pasang di OBS sekali, langsung hidup. Gratis selamanya.

Coba Sekarang Lihat Tutorial

Cara Pasang di OBS

Cukup 5 menit. Ikuti langkah-langkahnya.

1. Buka OBS Studio

Pastikan OBS sudah terinstall. Kalau belum, download gratis di obsproject.com.

2. Tambah Browser Source

  1. Di panel Sources (kiri bawah OBS), klik tombol +

  2. Pilih Browser dari menu

    Kalau tidak ada opsi Browser, update OBS ke versi terbaru.

3. Isi Properti Browser

PropertiIsi
URLhttps://ramaraihan12.github.io/chaBubbleTwitch/chat-bubble.html
Width400
Height600
PENTING: Hilangkan centang "Shutdown source when not visible" agar overlay tidak mati saat pindah scene.

4. Connect ke Channel

  1. Di panel Sources, klik kanan source "Chat Bubble" → pilih Interact

    Akan muncul jendela interaksi dengan kotak input dan tombol Connect.

  2. Ketik nama channel Twitch Anda

    Contoh: kalau URL Twitch Anda twitch.tv/fahri_gaming, isi dengan fahri_gaming. Tanpa @, tanpa spasi, tanpa https://.

  3. Klik Connect — tunggu sampai muncul "Connected: namachannel"

    Biasanya butuh 2-3 detik. Tombol akan berubah jadi Disconnect.

  4. Tutup jendela Interact. Selesai.

    Chat sekarang muncul sebagai bubble di pojok kanan bawah layar.
Tips: Setelah connect, posisi bubble bisa digeser dan di-resize seperti layer biasa di OBS. Drag langsung di preview.

Cara Ganti Channel

Kalau mau pindah ke channel Twitch lain.

  1. Klik kanan source → Interact

  2. Klik Disconnect

  3. Ketik nama channel baru → Connect

Troubleshooting

Chat tidak muncul

Pastikan ada orang yang chat di channel Anda. Coba chat sendiri dari HP atau akun lain. Kalau masih tidak muncul, Disconnect lalu Connect ulang.

"Gagal connect" atau "Connecting..." terus

Periksa nama channel sudah benar (tanpa @, tanpa spasi, tanpa https://). Cek koneksi internet. Coba tutup dan buka ulang OBS.

Bubble terpotong

Ukuran Browser Source di OBS terlalu kecil. Perbesar Width dan Height di properti source. Rekomendasi: Width 400, Height 600.

Harus connect ulang setiap restart OBS?

Saat ini ya. Setiap restart OBS, lakukan Interact → Connect kembali. Fitur auto-connect direncanakan di update berikutnya.
