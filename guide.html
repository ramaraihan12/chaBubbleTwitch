A. Apa Itu Chat Bubble Overlay?

Chat Bubble Overlay adalah tampilan chat Twitch kustom berbentuk gelembung (bubble) yang bisa dipasang di OBS Studio. Setiap kali ada yang chat di channel Anda, pesannya muncul sebagai bubble animasi di pojok kanan bawah layar. Tidak perlu login, tidak perlu token, tidak perlu bayar.

Tampilan bubble-nya seperti ini:

  • Atas bubble: Nama user (warna mengikuti warna Twitch user)
  • Bawah bubble: Isi pesan chat (termasuk emote)

B. Sebelum Mulai

Yang Anda butuhkan:

  • OBS Studio (download gratis di obsproject.com)
  • Koneksi internet stabil
  • Link overlay: https://ramaraihan12.github.io/chaBubbleTwitch/chat-bubble.html

C. Cara Pasang di OBS Studio

Langkah 1 — Buka OBS Studio
Langkah 2 — Di panel Sources (kiri bawah), klik tombol +
Langkah 3 — Pilih Browser dari menu dropdown
Kalau tidak ada opsi "Browser", pastikan OBS Anda versi terbaru. Download ulang dari obsproject.com.
Langkah 4 — Beri nama (misal "Chat Bubble"), centang "Make source visible", klik OK
Langkah 5 — Isi properti browser:

Height600
Custom CSS(kosongkan)

Langkah 6 — Klik OK. Source sudah terpasang.
PENTING: Pastikan "Shutdown source when not visible" TIDAK dicentang. Kalau dicentang, overlay akan mati setiap pindah scene.

D. Cara Connect ke Channel Twitch

Langkah 1 — Di panel Sources OBS, klik kanan source "Chat Bubble"
Langkah 2 — Pilih Interact

Akan muncul jendela baru. Di tengah jendela ada kotak input dan tombol Connect berwarna ungu.

Langkah 3 — Ketik nama channel Twitch Anda (tanpa spasi, tanpa @)

Contoh: kalau URL Twitch Anda twitch.tv/fahri_gaming, isi dengan fahri_gaming

Langkah 4 — Klik tombol Connect

Tunggu 2-3 detik. Setelah connect, akan muncul teks "Connected: namachannel" di pojok kiri atas.

Langkah 5 — Tutup jendela Interact (klik X)

Chat sekarang sudah aktif. Setiap orang yang chat di channel Anda akan muncul sebagai bubble.

Kalau status "Connecting..." berputar lama, cek koneksi internet. Twitch kadang butuh beberapa detik untuk menyambungkan.

E. Cara Ganti Channel

  1. Klik kanan source → Interact
  2. Klik Disconnect
  3. Ketik nama channel baru
  4. Klik Connect

F. Cara Ganti Posisi / Ukuran

Di OBS, Anda bisa menggeser dan mengubah ukuran bubble overlay seperti layer biasa:

  1. Klik source "Chat Bubble" di panel Sources
  2. Di preview OBS, drag tepinya untuk resize
  3. Drag tengahnya untuk pindah posisi

Rekomendasi: posisikan di pojok kanan bawah agar tidak menutupi gameplay.

G. Troubleshooting

Chat tidak muncul

  • Pastikan ada orang yang chat di channel Twitch Anda (coba chat sendiri dari HP)
  • Cek koneksi internet
  • Disconnect lalu Connect ulang

Tulisan "Gagal connect"

  • Periksa nama channel sudah benar (tanpa @, tanpa spasi, tanpa https://)
  • Coba tutup OBS, buka lagi, ulangi Langkah C dan D

Bubble terpotong atau aneh

  • Ukuran overlay di OBS terlalu kecil. Perbesar width dan height di properti Browser source.

Harus connect ulang setiap buka OBS?

  • Saat ini ya. Setiap OBS restart, lakukan Interact → Connect. Kedepannya akan dibuat fitur auto-connect.
